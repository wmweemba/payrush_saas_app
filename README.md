# BazaBooks

> Work. Invoice. Get paid.

Professional invoicing for freelancers and small businesses. Create invoices and quotes in under 2 minutes, share via WhatsApp or email, and track what's paid — from mobile or desktop.

## Features

- Invoice and quote creation with sequential numbering (INV-001, QT-001)
- Multi-currency support
- Client management
- Shareable public invoice links
- PDF generation with brand colours and logo
- WhatsApp, Telegram, and email sharing
- PWA — installable on mobile
- Dashboard with payment tracking

## Tech stack

Next.js 15 App Router · Better Auth · Drizzle ORM · PostgreSQL · Tailwind CSS v4 · Cloudflare R2 · Resend
